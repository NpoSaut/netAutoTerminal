Программа упаковки:
1) тип системы
2) тип блока
3) модификация блока
4) програмный модуль
5) Версия ПО (версия/подверсия)
6) Дата релиза ПО

1) тип системы            |
2) тип блока              |  может быть взято из файла с прошивкой
3) модификация блока      |
4) серийный номер блока
5) дата изготовления
6) Файл с прошивкой       |  может быть подобрана исходя из (1-3)
